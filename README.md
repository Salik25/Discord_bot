# Дискорд бот для управления канала
## Реализованы следующие функции
   
- [ ] Выдача роли при входе пользователя
- [ ] Отправка сообщения любому пользователю, а так же самому себе(для игры в мафию на канале)
- [ ] Удаление сообщений
- [ ] Kick and Ban
- [ ] Ответ на частые слова
- [ ] Mute and unmute user, а так же mute and unmute channel
- [ ] Игра крестики - нолики между двумя пользователями
